实现泛型`TupleToUnion<T>`，它覆盖元组的值与其值联合。

例如

```ts
type Arr = ['1', '2', '3']

const a: TupleToUnion<Arr> // expected to be '1' | '2' | '3'
```



```ts
type TupleToUnion<T extends unknown[]> = T[number]

type TupleToUnion<T extends any[]> = keyof {
     [O in T[number]]: O
}
```


`T[number]` 返回泛型数组的元素
