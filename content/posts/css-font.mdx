```
span {
        font-family: Helvetica,Arial,Alibaba PuHuiTi, Microsoft YaHei, PingFang SC, sans-serif;
        font-weight: 400;
}
```
先英文后中文，中文采用雅黑，Mac上使用苹方，不设置的话会用宋体字显示，奇丑无比。 最后`sans-serif` 保底。
